# TLC Taxi Data
The original TLC taxi data is extremely large: ~30 GB for 6 years (2020~2025). If you need to download the data, please run the second cell "Download TLC data" in the Data_preprocessing.ipynb. (You may need to run the cell for several times to continue downloading)
